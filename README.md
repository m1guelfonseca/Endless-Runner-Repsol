# 🏎️ Endless Runner Repsol

> Um jogo de corrida infinita desenvolvido em Unity, com temática Repsol, desenvolvido no âmbito da cadeira de **Tecnologias e Multimédia**.

---

## 📖 Sobre o Projeto

**Endless Runner Repsol** é um jogo estilo *endless runner* onde o jogador controla um carro com a icónica identidade visual da Repsol, desviando-se de obstáculos numa estrada que não tem fim. O objetivo é sobreviver o máximo de tempo possível e alcançar a maior pontuação.

O projeto foi desenvolvido em **Unity** com scripts escritos em **C#**, como trabalho prático da cadeira de Tecnologias e Multimédia.

---

## 🎮 Mecânicas de Jogo

- **Carro controlável** — o jogador controla lateralmente um carro com identidade visual Repsol
- **Obstáculos gerados proceduralmente** — o cenário e os obstáculos são gerados infinitamente à medida que o jogo avança
- **Velocidade progressiva** — o jogo acelera com o tempo, aumentando a dificuldade
- **Sistema de pontuação** — pontuação baseada na distância percorrida / tempo de sobrevivência
- **Game Over** — colisão com obstáculos termina a corrida

## 🚀 Como Executar

### Pré-requisitos

- [Unity Hub](https://unity.com/download) instalado
- Unity Editor (versão compatível com o projeto)
- Git

### Passos

1. **Clona o repositório:**
   ```bash
   git clone https://github.com/m1guelfonseca/Endless-Runner-Repsol.git
   ```

2. **Abre o Unity Hub** e clica em **"Add project from disk"**

3. **Seleciona a pasta** `Endless-Runner-Repsol/` clonada

4. **Abre o projeto** no Unity Editor

5. Na janela **Project**, navega até à cena principal em `Assets/Scenes/` e faz duplo clique para a abrir

6. Clica em **Play ▶** para iniciar o jogo

## 🎯 Controlos

| Tecla | Ação |
|---|---|
| `←` / `A` | Mover carro para a esquerda |
| `→` / `D` | Mover carro para a direita |
| `Espaço` | (se aplicável) Ação adicional |
